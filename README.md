# Monka's Graphics Optimizer

Retail World of Warcraft addon that adds a Game Menu button for quickly applying FPS-focused graphics settings.

## Features

- Adds Monka's Graphics Optimizer to the Escape menu.
- Applies a preset of performance-oriented graphics CVars.
- Refuses to change settings during combat lockdown.

## Installation

1. Download or clone this repository.
2. Place the `Monka's Graphics Optimizer` folder inside your `World of Warcraft\\_retail_\\Interface\\AddOns` directory.
3. Launch the game and enable the addon from the AddOns list.

## Publishing

This repository is structured so the addon folder itself is the repository root, which is compatible with GitHub-based release packaging workflows.
